INSE API Dictionary

Getting a post

GET

/forum?thread=[param]

Gets the post and all of the comments related to the given thread ID in JSON format

1 - The ID for the thread to get

Getting a Board

GET

/forum?board=[param]&order=[param]

Gets a collection of posts for a given board, ordered using the specified method

1 - The ID of the board from which to get a collection of posts

2 - The order in which the posts should be delivered

Create Post

POST

/forum?board=[param]

Creates a post on the given board using the information provided in the body of the POST

1 - The ID of the board in which to create the post

Create Comment

POST

/forum?thread=[param]&comment=[param]

Creates a comment in the given thread, or creates a reply to another comment if the comment parameter is fufilled. Comment information is stored in the body of the POST.

1 - The ID of the thread to make the comment in

2 (Optional) - The ID of the comment to make a reply to

Edit Comment

POST

/forum?thread=[param]&comment=[param]

Edits the content of a comment, updating it with information inside of the body of the POST

1 - The ID of the thread the comment is in

2 - The comment ID to edit

Edit Post

POST

/forum?board=[param]&thread=[param]

Edits the content of a given thread using the information provided by the POST.

1 - The ID of the board the thread is in

2 - The ID of the thread to edit

Like Comment

POST

/forum?thread=[param]&comment=[param]&status=like

Likes a given comment from a given thread

1 - The ID of the thread the comment is in

2 - The ID of the comment to like

Dislike Comment

POST

/forum?thread=[param]&comment=[param]&status=dislike

Dislikes a given comment from a given thread

1 - The ID of the thread the comment is in

2 - The ID of the comment to dislike

Like Post

POST

/forum?board=[param]&thread=[param]&status=like

Likes a given post from a given board

1 - The ID of the board the thread is in

2 - The ID of the thread to like

Dislike Post

POST

/forum?board=[param]&thread=[param]&status=dislike

Dislikes a given post from a given board

1 - The ID of the board the thread is in

2 - The ID of the thread to dislike

Mark Answer

POST

/forum?thread=[param]&comment=[param]&correct=true

Marks a given comment as the correct answer for a given thread

1 - The ID of the thread to mark the comment in

2 - The ID of the comment of which to mark as the correct answer

Create a reply to a comment

POST

/forum?thread=[param]&comment=[param]&status=reply

Creates a reply to a comment on a board using the ID of the board and the ID of the comment to reply to

1 - The ID of the board on which the comment is being replied to

2 - The ID of the comment to reply to

Reporting a post

POST

/forum/report?thread=[param]

Reports a given thread

1 - The ID of the thread to report

Report Comment

POST

/forum/report?thread=[param]&comment=[param]

Reports a given comment

1 - The ID of thread the comment is in

2 - The ID of the comment to report
